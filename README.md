# COINANT多页项目

> COINANT活动页，推广页，内嵌页

**技术栈**

* webpack2.0
* es6
* scss
* postcss

**项目配置项**
``` bash
# 项目初始化
yarn
npm install
# 项目启动
yarn start / yarn run dev
npm start / npm run dev

# 项目发布
yarn run build
npm run build
```

**项目部署**

- 测试: svn://192.168.92.101/webfiles/inline.coinant.io

**命名规范**

- 文件名：
单词用‘-’隔开，如：customer-service
**(文件名中严禁出现大写字字母和下划线)**

- 变量名：
使用小驼峰命名，如：customerService
